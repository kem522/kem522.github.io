About

After graduating from New York University, I volunteered with an organization that was eager to migrate from its out-of-date Ruby on Rails / MySQL site to a more robust and scalable site on a highly-customized e-commerce platform. They did not have a tech person on staff and so, despite having non coding experience, I offered to figure it out and make it happen. Jumping right in was a great way to learn html, CSS, and JavaScript! It was also a great way to discover that web development offers an ideal way for me to combine my interests in creating things, solving complex problems, working in teams, and making a difference. Web development is doubly-rewarding: at work, it provides me a way to contribute to the efforts of a company whose values I share, and in my free time, it lets me make a difference in the world by volunteering my skills to help nonprofits doing important work. Despite having spent the better part of a year preparing for and taking the LSAT, I researched coding bootcamps, and I just finished the Web program at General Assembly.

I am looking for a position in a team with a strong emphasis on continued learning, and who are making products that directly improve people’s lives.

Projects

Tetris

Github Heroku Skills used: JavaScript (ES6/ES7), HTML, CSS, jQuery
We were given one week to create a game using only JavaScript, HTML and CSS. My finished game had all the basic Tetris features including rotation, rotation at the edge of the game board resulting in the shape bouncing inwards to allow it to turn, line clearing, use of the keys to move side to side and speed up the fall, and a scoring system.

Basicgram

Github Heroku Skills used: HTML, CSS, MVC, RESTful APIs, MongoDB with Mongoose, Node.js, EJS
This project helped us to understand server side architecture, the MVC model, and user authentication by way of sessions. We had the choice of three types of app; I chose the instagram clone as I thought it would would give me the best opportunity to better understand embedded and referenced data.

MateDate

Github Heroku Skills used: HTML, CSS, AngularJS, MVC, RESTful APIs, MongoDB with Mongoose, Node.js, JWT
For this project, my partner and I built a site that allows users to create an event, invite their friends and then use a polling system to decide where to eat using Yelp’s API. It also uses Google’s Maps and Places APIs to provide the user with a more informative and easier experience.

Nostalgiafy

Github Heroku Skills used: HTML, SASS, React, MVC, RESTful APIs, MongoDB with Mongoose, Node.js, JWT
This is a MERN stack app that allows users to put together a playlist of tracks from certain eras of their lives - such as during university or in their 20s. The site fetches the songs from the Billboard Year-End Hot 100 Singles on Wikipedia, then it uses the YouTube API to populate the iframes. Using this same API, the visitor can also sign in with Google and create a playlist on YouTube, itself.

Voluntary Work

Changing The Present

The New York Times called this nonprofit, "an Amazon.com of the nonprofit world." It channels to nonprofits some of the $450 billion Americans spend each year on birthday, wedding, and holiday presents. It does that by making the experience of a donation, given in a friend's name, feel like a rewarding gift and a great alternative to buying yet more "stuff." Features include tangible charitable gifts, personalized printed greeting cards, wish lists and registries, wedding favors, bulk orders, and gift cards.

Free the Textbooks

FreeTheTextbooks is the focal point for a movement to lower the cost of college textbooks for 21 million college students.

Bespoke sites for larger nonprofits

This is a website to let a client nonprofit capture money normally spent on presents. Like ChangingThePresent.org, it has such features as tangible charitable gifts, personalized printed greeting cards, wish lists and registries, wedding favors, bulk orders, and gift cards.
