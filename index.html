About

Whilst studying, I started volunteering for a nonprofit. I was assigned to help update the website despite having no background in coding. Very quickly I realized that web development was where my interests, abilities and career aspirations could come together. The flexibility of web development suggests to me that I could easily work for a company who shares my values and use my free time to volunteer my skills for nonprofits who need the help. Despite spending a year preparing for and taking the LSAT, I started researching coding bootcamps which led me to GA.

Ideally I would like to work as a Front-End Developer in a team with a strong emphasis on continued learning to make products that directly improve people’s lives.

Projects

Tetris

Github Heroku Skills used: JavaScript (ES6/ES7), HTML, CSS, jQuery
We were given one week to create a game using only JavaScript, HTML and CSS. My finished game had all the basic Tetris features including rotation, rotation at the edge of the game board resulting in the shape bouncing inwards to allow it to turn, line clearing, using the keys to move side to side and speed up the fall, and a scoring system.

Basicgram

Github Heroku Skills used: HTML, CSS, MVC, RESTful APIs, MongoDB with Mongoose, Node.js, EJS
This project helped us to understand server side architecture, the MVC model and user authentication by way of sessions. We had the choice of three types of app, I chose the instagram clone as I thought it would would give me a lot of opportunity to better understand embedded and referenced data.

MateDate

Github Heroku Skills used: HTML, CSS, AngularJS, MVC, RESTful APIs, MongoDB with Mongoose, Node.js, JWT
For this project, both my partner and I built a site that allows users to create an event, invite their friends and then use a polling system to decide where to eat using Yelp’s API. It also uses Google’s Maps and Places APIs to provide the user with a more informative and easier experience.

Nostalgiafy

Github Heroku Skills used: HTML, SASS, React, MVC, RESTful APIs, MongoDB with Mongoose, Node.js, JWT
This is a MERN stack app that allows users to put together a playlist of tracks from certain eras of their lives - such as during university or in their 20s. The songs are fetched from the Billboard Year End Hot 100 Singles on Wikipedia, then it uses the YouTube API to populate the iframes. Using this same API, you can also sign in with Google and create a playlist on YouTube itself.
